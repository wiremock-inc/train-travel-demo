# Currency

Three-letter [ISO currency code](https://www.iso.org/iso-4217-currency-codes.html), in lowercase.

## Example Usage

```typescript
import { Currency } from "train-travel/models/operations";

let value: Currency = "nok";
```

## Values

```typescript
"bam" | "bgn" | "chf" | "eur" | "gbp" | "nok" | "sek" | "try"
```
